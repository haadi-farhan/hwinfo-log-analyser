# HWiNFO Log Analyser

### View online at: https://hwinfo-log-analyser.vercel.app/ *(Ctrl + Left Click to open in a new tab)
---

A sleek, modern web application designed to allow users to easily explore, visualise and analyse log data generated by HWiNFO. Built as the final capstone project for the **Google AI Professional Certificate** on Coursera. The app can run completely offline and does not make use of the Gemini API, so no API key is required to use it.
<br>
<br>

## Why This Project?
Existing HWiNFO log viewers often feature outdated, cluttered interfaces. This motivated me to build this application to solve that issue by providing a stylish and minimal user interface.

## Key Features
* **Intuitive Metric Selection:** Easily navigate through hundreds of sensors using the built-in search function and categorised menus.
* **Interactive Data Visualisation:** View your log data on clean, responsive graphs.
* **Time Snapshots:** Click anywhere on a graph to instantly view an organised snapshot of all system metrics at that exact moment.
* **Limits & Throttling Highlights:** Toggle to show/hide system limits and highlight regions where performance throttling occurred.
* **Metric Summary Tabs:** Convenient summary tabs located beneath graphs provide quick insights into minimum, maximum, and average values.
* **100% Client-Side Processing:** All data parsing and visualisation happens locally in your browser. Uploaded log files are never sent to a server.

## Development Details
* Built using Google AI Studio.
* **Development Time**: Rapid, took ~4 hours (including time spent navigating the free-tier rate limits).
* **Development Approach**: AI-assisted development ('vibe coding'). This was my first time using an AI-oriented IDE to build an app, and through the course I learned advanced prompting techniques and how to make use of tools in Google AI Studio to effectively streamline the build process.


## Current Status & Testing
* **Testing Scope**: Seems stable and bug-free from testing using real world logs from my system (Intel CPU and NVIDIA GPU). 
* *Note: Extensive cross-hardware testing has not been conducted yet. Contributions and bug reports are welcome.*


## To Run Locally:
**Prerequisites:**  Node.js
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
